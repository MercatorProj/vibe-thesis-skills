# 术语标准化参考

## 车辆动力学术语

### 基本参数
- **接地压力**: Ground Pressure, 单位: kN/m²
- **车辆质量**: Vehicle Mass, 单位: kg
- **轮胎/履带接地面积**: Contact Area, 单位: m²
- **车辆圆锥指数**: Vehicle Cone Index (VCI)
- **土壤圆锥指数**: Cone Index (CI)

### 动力性能
- **最大爬坡能力**: Maximum Climbing Ability, 单位: 度(°)或弧度(rad)
- **最大牵引力**: Maximum Tractive Force, 单位: kN
- **行驶阻力**: Running Resistance, 单位: kN
- **滚动阻力**: Rolling Resistance, 单位: kN
- **空气阻力**: Air Resistance, 单位: kN

## 土壤力学术语

### 土壤参数
- **内聚模数**: Cohesion Modulus, 单位: kN/m^(n+1)
- **内摩擦系数**: Internal Friction Coefficient, 单位: kN/m^(n+2)
- **变形指数**: Deformation Index, 无量纲
- **土壤含水率**: Soil Moisture Content, 单位: %或小数

### 沉陷相关
- **沉陷深度**: Sinkage Depth, 单位: m
- **沉陷阻力**: Sinkage Resistance, 单位: kN
- **临界沉陷深度**: Critical Sinkage Depth, 单位: m

## 路径规划术语

### 算法相关
- **A*算法**: A* Algorithm
- **启发函数**: Heuristic Function
- **代价函数**: Cost Function
- **曼哈顿距离**: Manhattan Distance
- **欧几里得距离**: Euclidean Distance

### 路径属性
- **路径长度**: Path Length, 单位: m
- **路径代价**: Path Cost, 无量纲或时间单位
- **通行性**: Passability, 二值(0/1)或连续值
- **坡度**: Slope, 单位: 度(°)或百分比(%)

## 调度优化术语

### VRP相关
- **车辆路径问题**: Vehicle Routing Problem (VRP)
- **取送问题**: Pickup and Delivery Problem (PDP)
- **容量约束**: Capacity Constraint
- **时间窗约束**: Time Window Constraint
- **配送中心**: Distribution Center
- **需求点**: Demand Point

### 优化方法
- **精确算法**: Exact Algorithm
- **启发式算法**: Heuristic Algorithm
- **元启发式算法**: Metaheuristic Algorithm
- **遗传算法**: Genetic Algorithm (GA)
- **模拟退火算法**: Simulated Annealing (SA)
- **禁忌搜索**: Tabu Search (TS)

## 地理信息术语

### 数据类型
- **数字高程模型**: Digital Elevation Model (DEM)
- **数字表面模型**: Digital Surface Model (DSM)
- **栅格数据**: Raster Data
- **矢量数据**: Vector Data

### 空间分析
- **空间叠加分析**: Spatial Overlay Analysis
- **缓冲区分析**: Buffer Analysis
- **邻域分析**: Neighborhood Analysis
- **坡度分析**: Slope Analysis
- **坡向分析**: Aspect Analysis

## 地质灾害术语

- **滑坡**: Landslide
- **泥石流**: Debris Flow
- **冻土融沉**: Permafrost Thaw Settlement
- **地质灾害风险评估**: Geological Hazard Risk Assessment
- **高风险区域**: High-risk Area
- **中风险区域**: Medium-risk Area
- **低风险区域**: Low-risk Area

## 实验验证术语

### 性能指标
- **准确率**: Accuracy
- **精确率**: Precision
- **召回率**: Recall
- **F1分数**: F1 Score
- **平均绝对误差**: Mean Absolute Error (MAE)
- **均方根误差**: Root Mean Square Error (RMSE)

### 对比分析
- **基准算法**: Baseline Algorithm
- **对比实验**: Comparative Experiment
- **性能提升**: Performance Improvement
- **计算时间**: Computation Time, 单位: s或ms
- **收敛速度**: Convergence Rate

## 学术写作常用表达

### 引用格式
- "参考...的研究" → "参考...等(年份)的研究"
- "根据...提出的方法" → "根据...等(年份)提出的方法"
- "本文采用...算法" → "本文采用...等(年份)提出的...算法"

### 逻辑连接词
- "首先...其次...最后" → "首先...其次...最终"
- "因此" → "因此" / "综上所述" / "基于此"
- "另外" → "此外" / "同时"
- "但是" → "然而" / "不过"

### 程度副词
- "很" → "显著" / "明显" / "充分"
- "非常" → "高度" / "极为"
- "比较" → "相对" / "较为"

### 动词选择
- "使用" → "采用" / "应用" / "运用"
- "得到" → "获得" / "得出" / "取得"
- "认为" → "认为" / "指出" / "表明"
- "说明" → "阐述" / "说明" / "论证"
