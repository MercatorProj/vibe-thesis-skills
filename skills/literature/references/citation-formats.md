# 引用格式详细说明

## 概述

本文档详细介绍LaTeX中常用的文献引用格式及其使用方法。

## 基本引用格式

### 1. 基本引用

**格式**: `\cite{文献简称}`

**示例**:
```latex
\cite{BHO2021_动态车辆路径问题综述}
```

**输出**:
```
[1]
```

### 2. 多项引用

**格式**: `\cite{文献简称1, 文献简称2, 文献简称3}`

**示例**:
```latex
\cite{BHO2021_动态车辆路径问题综述, Mor2022_多商品车辆路径问题, Zhang2020_电动汽车路径规划模糊优化}
```

**输出**:
```
[1, 2, 3]
```

### 3. 带页码引用

**格式**: `\cite[p. 页码]{文献简称}`

**示例**:
```latex
\cite[p. 10]{BHO2021_动态车辆路径问题综述}
```

**输出**:
```
[1, p. 10]
```

### 4. 带章节引用

**格式**: `\cite[Chapter 章节]{文献简称}`

**示例**:
```latex
\cite[Chapter 3]{BHO2021_动态车辆路径问题综述}
```

**输出**:
```
[1, Chapter 3]
```

### 5. 带页码范围引用

**格式**: `\cite[pp. 页码范围]{文献简称}`

**示例**:
```latex
\cite[pp. 10-20]{BHO2021_动态车辆路径问题综述}
```

**输出**:
```
[1, pp. 10-20]
```

### 6. 带备注引用

**格式**: `\cite[备注]{文献简称}`

**示例**:
```latex
\cite[参见第10页]{BHO2021_动态车辆路径问题综述}
```

**输出**:
```
[1, 参见第10页]
```

## 高级引用格式

### 1. 作者年份引用

**格式**: `\citep{文献简称}`

**示例**:
```latex
\citep{BHO2021_动态车辆路径问题综述}
```

**输出**:
```
(BHO et al., 2021)
```

### 2. 作者年份多项引用

**格式**: `\citep{文献简称1, 文献简称2, 文献简称3}`

**示例**:
```latex
\citep{BHO2021_动态车辆路径问题综述, Mor2022_多商品车辆路径问题, Zhang2020_电动汽车路径规划模糊优化}
```

**输出**:
```
(BHO et al., 2021; Mor et al., 2022; Zhang et al., 2020)
```

### 3. 作者年份带页码引用

**格式**: `\citep[p. 页码]{文献简称}`

**示例**:
```latex
\citep[p. 10]{BHO2021_动态车辆路径问题综述}
```

**输出**:
```
(BHO et al., 2021, p. 10)
```

### 4. 文本内引用

**格式**: `\citet{文献简称}`

**示例**:
```latex
\citet{BHO2021_动态车辆路径问题综述}指出，动态车辆路径问题是车辆路径规划领域的研究热点之一。
```

**输出**:
```
BHO et al. (2021)指出，动态车辆路径问题是车辆路径规划领域的研究热点之一。
```

### 5. 文本内多项引用

**格式**: `\citet{文献简称1, 文献简称2, 文献简称3}`

**示例**:
```latex
\citet{BHO2021_动态车辆路径问题综述, Mor2022_多商品车辆路径问题, Zhang2020_电动汽车路径规划模糊优化}分别对动态车辆路径问题、多商品车辆路径问题和电动汽车路径规划问题进行了深入研究。
```

**输出**:
```
BHO et al. (2021), Mor et al. (2022)和Zhang et al. (2020)分别对动态车辆路径问题、多商品车辆路径问题和电动汽车路径规划问题进行了深入研究。
```

## 引用格式选择

### 1. 直接引用

**适用场景**: 直接引用文献中的观点或数据

**格式**: `\cite{文献简称}`

**示例**:
```latex
动态车辆路径问题（DVRP）是指在需求随时间变化且存在不确定因素的情况下，对车辆路径进行实时规划和调整的问题\cite{BHO2021_动态车辆路径问题综述}。
```

### 2. 间接引用

**适用场景**: 间接引用文献中的观点或数据

**格式**: `\citep{文献简称}`

**示例**:
```latex
近年来，动态车辆路径问题得到了广泛的研究和应用\citep{BHO2021_动态车辆路径问题综述}。
```

### 3. 文本内引用

**适用场景**: 在文本中直接提到作者

**格式**: `\citet{文献简称}`

**示例**:
```latex
\citet{BHO2021_动态车辆路径问题综述}指出，动态车辆路径问题是车辆路径规划领域的研究热点之一。
```

### 4. 多项引用

**适用场景**: 同时引用多篇文献

**格式**: `\cite{文献简称1, 文献简称2, 文献简称3}`

**示例**:
```latex
动态车辆路径问题和多商品车辆路径问题都是车辆路径规划领域的研究热点\cite{BHO2021_动态车辆路径问题综述, Mor2022_多商品车辆路径问题}。
```

## 引用格式转换

### 1. 数字引用转换为作者年份引用

**原格式**:
```latex
动态车辆路径问题得到了广泛的研究和应用\cite{BHO2021_动态车辆路径问题综述}。
```

**转换后**:
```latex
动态车辆路径问题得到了广泛的研究和应用\citep{BHO2021_动态车辆路径问题综述}。
```

### 2. 作者年份引用转换为数字引用

**原格式**:
```latex
动态车辆路径问题得到了广泛的研究和应用\citep{BHO2021_动态车辆路径问题综述}。
```

**转换后**:
```latex
动态车辆路径问题得到了广泛的研究和应用\cite{BHO2021_动态车辆路径问题综述}。
```

### 3. 文本内引用转换为数字引用

**原格式**:
```latex
\citet{BHO2021_动态车辆路径问题综述}指出，动态车辆路径问题是车辆路径规划领域的研究热点之一。
```

**转换后**:
```latex
文献\cite{BHO2021_动态车辆路径问题综述}指出，动态车辆路径问题是车辆路径规划领域的研究热点之一。
```
