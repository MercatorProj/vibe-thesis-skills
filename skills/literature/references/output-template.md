# 文献阅读总结输出模板

按任务复杂度在以下模板中选择一种，默认优先用“逐篇摘要模板 + 多篇综合模板”。

## 一、逐篇摘要模板

```markdown
## 文献 1

### 1. 元数据
- 标题：
- 作者：
- 年份：
- 来源（期刊/会议/学位论文/报告）：
- DOI/标识：
- 关键词：

### 2. 文章主旨
- 用 3-5 句话说明研究问题、研究动机、核心思路与主要贡献。

### 3. 方法与实验要点
- 研究对象/场景：
- 核心方法/模型/算法：
- 数据或案例：
- 评价指标：
- 关键结果：

### 4. 与本文相关内容
- 与本课题直接相关的概念、方法、数据或结论：
- 可用于哪一章/哪一类表述：
- 可作为哪类引用：背景、相关工作、方法依据、实验对比、案例支撑。

### 5. 启发性信息
- 启发 1：
- 启发 2：
- 启发 3：

### 6. 局限性或使用注意
- 作者指出的局限：
- 从本文角度看需要注意的问题：
```

## 三、适配论文写作时的附加栏目

若用户明确要服务于学位论文，可额外补充：

- **可插入绪论/综述的论点**
- **可支撑方法设计的依据**
- **可作为实验或案例对比的来源**
- **建议检索的补充关键词**

## 四、质量检查

- 元数据是否有缺失或不确定项，并已明确标注？
- “作者结论”和“我的归纳/启发”是否分开？
- 与本文相关性的判断是否具体，而不是泛泛地说“有帮助”？
- 是否给出了可执行的启发，而不只是复述原文？
