# Image Description

## Overview

本技能为学位论文或期刊论文中的**插图**提供符合规范的叙述方式，采用「正文为核心，图片为佐证」的学术写作原则：将图示信息融入论证链条，作为支持论点的证据，并与全篇逻辑一致。具体学科主题（如交通、医学、材料等）由用户给定，技能不假设某一固定课题。

## Workflow

### 图片描述工作流程

1. **研究主题分析**：理解论文研究主题和核心论点
2. **图片功能定位**：明确图片在论文中的作用和佐证目的
3. **内容提取整合**：提取图片关键信息，与正文内容整合
4. **叙述框架构建**：以正文论述为主线，图片信息为支撑
5. **学术语言润色**：确保描述符合学术规范和论文风格

## Image Types and Integration Patterns

### 1. 地图类图片
- **研究区域地图**：作为研究背景的佐证，融入区域概况叙述中
- **空间分析/场量图**：作为模型或观测结果的实证，支持空间分布结论
- **路径或方案图**：作为算法或决策结果的展示，验证方法有效性
- **调度或网络图**：作为优化方案的可视化，支持策略论证

### 2. 流程图类图片
- **算法流程图**：作为算法设计的说明，融入算法原理叙述中
- **数据处理流程图**：作为数据处理方法的展示，支持数据可靠性论证
- **求解流程图**：作为模型求解过程的说明，支持求解方法合理性
- **验证流程图**：作为实验设计的展示，支持验证方法科学性

### 3. 数据图表类图片
- **性能对比图**：作为算法或模型性能的实证，支持比较性论点
- **优化结果图**：作为优化效果的展示，验证目标函数改进
- **敏感性分析图**：作为参数影响的分析，支持模型鲁棒性论证
- **成本或资源图**：作为成本构成的展示，支持可行性分析

### 4. 概念示意图类图片
- **模型架构图**：作为模型设计的说明，融入模型构建叙述中
- **系统概念图**：作为系统设计的展示，支持系统架构合理性
- **算法原理图**：作为算法原理的说明，支持算法创新点论证
- **问题定义图**：作为问题边界的展示，支持研究范围合理性

## Academic Language Standards

### 学术语言规范

1. **正文为主**：以论文论述为主线，图片信息为佐证材料
2. **逻辑连贯**：描述与论文整体逻辑保持一致，自然融入叙述
3. **论点支撑**：图片信息直接支持论文论点，强化论述说服力
4. **科学准确**：使用准确的学术术语，避免模糊表达
5. **简洁明了**：语言简洁，重点突出，避免冗余信息
6. **客观中立**：保持学术客观性，避免主观评价
7. **格式规范**：符合学位论文或目标期刊的图片引用和描述格式

## Usage

### 使用方法

当用户需要描述论文中的图片时，请按照以下步骤操作：

1. **提供研究背景**：简述图片所在章节的研究内容和核心论点
2. **说明图片作用**：指出图片在论文中的佐证目的和预期作用
3. **描述图片内容**：提供图片的类型和主要内容概述
4. **技能将根据上述信息生成以正文为核心、图片为佐证的学术描述**

## Notes

### 注意事项

1. **正文为主原则**：始终以论文论述为主线，图片信息作为佐证材料融入叙述
2. **论点支撑**：确保图片描述直接支持论文论点，强化论述说服力
3. **逻辑连贯**：描述应与论文整体逻辑保持一致，自然融入叙述流程
4. **信息准确**：图片描述应与图片内容严格对应，避免虚构信息
5. **重点突出**：对于复杂图片，重点描述与论文论点相关的关键信息
6. **格式规范**：符合学位论文或期刊的图片引用格式，包括图号、图名和描述
7. **学术客观**：使用第三人称与规范术语，避免主观评价
